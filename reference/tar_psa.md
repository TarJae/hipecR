# Peritoneal Surface Area Calculation

This function calculates 1 the given height and weight. It's used as a
factor for further calculations.

## Usage

``` r
tar_psa(height, weight)
```

## Arguments

- height:

  A numeric value representing the height in cm.

- weight:

  A numeric value representing the weight in kg.

## Value

A numeric value representing 1

## Examples

``` r
tar_psa(180, 80) # Expected output: 199.6421 cm^2
#> [1] 199.6421
```
