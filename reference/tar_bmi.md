# Calculating BMI (kg/m^2)

Calculating BMI (kg/m^2)

## Usage

``` r
tar_bmi(weight, height)
```

## Arguments

- weight:

  Numeric, kg

- height:

  Numeric, cm

## Value

body mass index BMI

## Examples

``` r
tar_bmi(weight = 100, height = 190)
#> [1] 27.70083
```
